# HW Engineer Portfolio MVP PRD

## 핵심 정보

**목적**: Hardware 엔지니어가 Notion에서 프로젝트를 작성하면 자동으로 반영되는 포트폴리오 웹사이트 제공
**사용자**: HW 엔지니어 본인 (콘텐츠 관리) + 채용담당자/동료 엔지니어 (포트폴리오 열람)

## 사용자 여정

```
1. 홈 페이지
   ↓ 프로젝트 둘러보기 클릭 / 헤더 메뉴 클릭

2. 프로젝트 목록 페이지
   ↓ 카테고리 필터 선택 또는 검색어 입력
   [카테고리 선택] → 필터링된 프로젝트 목록
   [검색어 입력] → 검색 결과 목록
   ↓ 프로젝트 카드 클릭

3. 프로젝트 상세 페이지
   ↓ 뒤로가기 또는 다른 프로젝트 탐색

4. 프로젝트 목록 페이지로 복귀
```

## 기능 명세

### 1. MVP 핵심 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|-------------|------------|
| **F001** | Notion API 연동 | Notion 데이터베이스에서 프로젝트 데이터를 조회하고 페이지 콘텐츠를 가져옴 | 전체 서비스의 데이터 소스 | 프로젝트 목록 페이지, 프로젝트 상세 페이지 |
| **F002** | 프로젝트 목록 표시 | 썸네일, 제목, 카테고리, 태그를 카드 형태로 목록 표시 | 포트폴리오 탐색의 핵심 | 프로젝트 목록 페이지 |
| **F003** | 프로젝트 상세 표시 | Notion 페이지 본문 블록(텍스트, 이미지, 코드, 임베드/동영상 등)을 렌더링 | 프로젝트 내용 전달의 핵심 | 프로젝트 상세 페이지 |
| **F004** | 카테고리 필터 | 프로젝트를 카테고리별로 필터링하여 표시 | HW 프로젝트 분류 탐색 | 프로젝트 목록 페이지 |
| **F005** | 프로젝트 검색 | 제목/태그 기반 키워드 검색 | 특정 프로젝트 빠른 탐색 | 프로젝트 목록 페이지 |
| **F006** | 반응형 레이아웃 | 모바일/태블릿/데스크톱 대응 반응형 UI | 다양한 기기에서 포트폴리오 열람 | 홈 페이지, 프로젝트 목록 페이지, 프로젝트 상세 페이지 |

## 페이지별 상세 기능

### 홈 페이지

> **구현 기능:** `F006` | **메뉴 위치:** 헤더 로고 / 홈 메뉴

| 항목 | 내용 |
|------|------|
| **역할** | 포트폴리오 랜딩 페이지 - 엔지니어 소개 및 프로젝트 탐색 유도 |
| **진입 경로** | 사이트 첫 방문, 헤더 로고 클릭, 홈 메뉴 클릭 |
| **사용자 행동** | 엔지니어 소개 확인, 프로젝트 둘러보기 버튼 클릭 |
| **주요 기능** | - 히어로 섹션 (이름, 직함, 한 줄 소개)<br>- 간략한 자기소개 영역<br>- **프로젝트 둘러보기** 버튼<br>- 반응형 레이아웃 |
| **다음 이동** | 프로젝트 둘러보기 클릭 → 프로젝트 목록 페이지 |

### 프로젝트 목록 페이지

> **구현 기능:** `F001`, `F002`, `F004`, `F005`, `F006` | **메뉴 위치:** 헤더 프로젝트 메뉴

| 항목 | 내용 |
|------|------|
| **역할** | 전체 프로젝트를 카드 그리드로 표시하고 필터/검색 제공 |
| **진입 경로** | 헤더 프로젝트 메뉴 클릭, 홈에서 프로젝트 둘러보기 클릭 |
| **사용자 행동** | 프로젝트 카드 탐색, 카테고리 필터 선택, 검색어 입력, 카드 클릭하여 상세 이동 |
| **주요 기능** | - Notion DB에서 Published 상태 프로젝트 조회 (F001)<br>- 썸네일/제목/카테고리/태그가 포함된 카드 그리드 (F002)<br>- 카테고리별 필터 탭 또는 버튼 (F004)<br>- 키워드 검색 입력 필드 (F005)<br>- 반응형 그리드 레이아웃 (F006) |
| **다음 이동** | 카드 클릭 → 프로젝트 상세 페이지 |

---

### 프로젝트 상세 페이지

> **구현 기능:** `F001`, `F003`, `F006` | **메뉴 위치:** 프로젝트 목록에서 카드 클릭

| 항목 | 내용 |
|------|------|
| **역할** | 개별 프로젝트의 전체 내용을 Notion 블록 기반으로 렌더링 |
| **진입 경로** | 프로젝트 목록에서 카드 클릭 |
| **사용자 행동** | 프로젝트 설명/이미지/동영상 열람, 스크롤하여 전체 내용 확인 |
| **주요 기능** | - 프로젝트 메타 정보 표시: 제목, 카테고리, 태그 (F001)<br>- Notion 본문 블록 렌더링: 텍스트, 이미지, 코드, 리스트 등 (F003)<br>- YouTube/동영상 임베드 렌더링 (F003)<br>- **목록으로 돌아가기** 버튼<br>- 반응형 레이아웃 (F006) |
| **다음 이동** | 목록으로 돌아가기 → 프로젝트 목록 페이지 |

| 필드 | 설명 | Notion 속성 타입 |

| Title | 프로젝트 제목 | Title |
| Category | 프로젝트 카테고리 (PCB Design, Firmware, Testing 등) | Select |
| Tags | 기술 태그 (Arduino, FPGA, Oscilloscope 등) | Multi-select |

| Status | 진행 상태 (Completed, In Progress 등) | Select |
